Project Overview

This project analyzes sales performance, customer behavior, and music trends using the Chinook Database.

The goal is to generate business insights and recommendations that can help a digital music store improve revenue, customer retention, and product strategy.

Objectives

Analyze revenue across countries, cities, and time
Identify high-value customers and purchasing behavior
Evaluate top-performing music genres, artists, and tracks
Assess employee contribution to overall revenue
Provide actionable business recommendations

Tools & Technologies
SQL (SQLite) – Data querying and analysis
Python (Pandas) – Data extraction and manipulation
Google Colab / Jupyter Notebook – Analysis environment
Matplotlib / Pandas Plotting – Data visualization

Dataset Description
The Chinook database simulates a digital music store and includes:
Customers
Invoices
Invoice_Items
Tracks
Albums
Artists
Genres
Employees

Key Business Questions

Revenue Analysis
Which countries and cities generate the most revenue?
What is the average invoice value per country?
Which year recorded the highest sales?

Customer Analysis
Who are the top spending customers?
Which countries have the most customers?
How does spending vary across regions?

Product & Sales Analysis
Which music genres and artists are most popular?
Which tracks generate the highest sales?
Which genres generate the most revenue?

Employee Performance
Which employee supports the highest-value customers?
Which employee contributes most to total revenue?

Key Insights
Revenue is concentrated in a few high-performing countries, indicating strong regional demand.
A small group of customers contributes a large portion of total revenue.
Certain music genres consistently outperform others in both sales and revenue.
Top-selling artists significantly influence overall business performance.

Business Recommendations
Focus marketing efforts on high-revenue countries to maximize returns
Implement loyalty programs for high-spending customers
Promote top-performing genres and artists to boost sales
Expand into emerging markets to diversify revenue sources
Strengthen customer engagement strategies to increase average spending

Project Workflow
Data Exploration
Examined tables and relationships within the database

SQL Analysis
Wrote queries to answer business questions
Used aggregation, filtering, and joins

Data Processing
Loaded SQL results into Python using Pandas

Visualization
Created charts to highlight key trends

Insights & Recommendations
Translated data findings into business decisions

Conclusion

This project demonstrates how SQL and Python can be used together to perform end-to-end business analysis — from querying raw data to generating actionable insights.
